## How to Obtain DFI {#how-to-obtain-dfi}

To obtain DFI, there are three main methods.

- Bake
- Crypto Factor Bridge
- Exchanges

### Bake {#Bake}

https://bake.io/

### Crypto Factor Bridge {#Bridge}

With QuickSwap, a user with tokens on Polygon can sell them to buy cDFI.
Using the Crypto Factor bridge at [https://cassets.crypto-factor.io/bridge](https://cassets.crypto-factor.io/bridge), users can bridge cDFI (Crypto Factor DFI) on Polygon to DeFiMetaChain, and then swap it to DFI as necessary. 

### Exchanges {#Exchanges}

- Bitrue ([DFI-USDT](https://www.bitrue.com/trade/dfi_usdt))
